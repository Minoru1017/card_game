## 2. 五款作品的「卡牌技能／效果」呈現方式

### 2.1 總表（與本作 A／B／C 對位）

| 作品 | 玩家何時看到「技能」 | 文字量級 | 是否「先無後有」解鎖規則 | 接近本作階段 |
|------|----------------------|----------|--------------------------|--------------|
| **Pokémon TCG Pocket** | 拿到卡即見**改寫後**的攻擊／能力（較實體版精簡） | 中～短（行動選項少） | **否**（設計師預先簡化卡池，非玩家進度解鎖） | 像「全域 A→B」：一開始就給**簡化版 B**，不是 A |
| **月圆之夜** | 選牌／獲牌當下即見**完整效果** | 中（Roguelike 單卡說明） | **否**（進 Run 後逐步拿牌，但每張卡本身不鎖條文） | 偏 **C**（單卡層級一次給齊） |
| **MARVEL SNAP** | 抽到即知 **On Reveal／Ongoing** 等一行機制 | **極短**（關鍵字可點） | **否**；Mastery **不解鎖規則** | 常態 = **B**；Mastery = 外觀（非 B/C） |
| **Slay the Spire** | 獎勵三選一即見**全文 + 工具提示** | 中～長 | **否** | 標準 **C** + UI 輔助 |
| **Balatro** | 購買 Joker／道具當下即見**全文** | 短～中 | **否** | 基底規則極簡（像 **A 的世界觀**），成長靠**新卡全文**（像 **C 元件**） |

### 2.2 分款摘要

#### （亞洲・商業化）Pokémon TCG Pocket

| 維度 | 內容 |
|------|------|
| **呈現** | 20 張牌組、自動能量、勝利條件改為擊倒 3 隻；**多數卡能力相對實體版刪減或改寫** |
| **玩家回饋（熱門相關）** | **正面**：collector 與新手願意「第一次真的打牌」；被媒體形容為把 TCG 簡化到可與快節奏產品競爭。**負面**：多貨幣、內容量大，首週仍覺得「confusing」 |
| **與 A/B/C 距離** | 熱門來自**全域簡化**，不是「同一張卡先 A 再 B 再 C」。與本作提案**路線不同**，但**目標一致**（降低門檻） |

#### （亞洲・獨立）月圆之夜

| 維度 | 內容 |
|------|------|
| **呈現** | Roguelike：**每張卡進牌組即帶完整效果**；複雜度來自「這局拿到什麼組合」，不是「這張卡還沒解鎖說明」 |
| **玩家回饋（熱門相關）** | **正面**：美術、童話 IP、重玩性、買斷良心。**負面**：後期 DLC／模式多後學習成本上升（仍是一次給齊規則，非分階鎖條文） |
| **與 A/B/C 距離** | 成功證明**華語市場願意為「卡牌 + Roguelike」付費**；但未驗證「熟練度鎖技能文字」 |

#### （非亞洲・商業化）MARVEL SNAP

| 維度 | 內容 |
|------|------|
| **呈現** | 12 張牌組、單局極短；卡面**一行能力** + 關鍵字；**Character Mastery（2025）** = 刷 Variant／Split 拿**閃光、Finish、表情**，**不改變對戰規則** |
| **玩家回饋（熱門相關）** | **正面**：快、爽、讀得懂、適合手機。**Mastery 負面**：進度綁**課金／抽 Variant** 而非「玩得熟」；池子被稀釋、無法自由混搭 Finish／Flare（社群常見抱怨） |
| **與 A/B/C 距離** | 有「Mastery」字樣但**不是本作意義的 B/C 規則解鎖**；若論文引用需**明確區分**，避免評審以為已有先例 |

#### （非亞洲・獨立）Slay the Spire

| 維度 | 內容 |
|------|------|
| **呈現** | **一開始就給完整卡面文字**；複雜度靠 Synergy 與路線選擇累積；工具提示降低誤讀 |
| **玩家回饋（熱門相關）** | **正面**：「易上手難精通」、每局不同、說明清楚。**負面**：少數卡敘述仍歧義；高難度需學習曲線（玩家接受，因為買斷 Roguelike 預期） |
| **與 A/B/C 距離** | 熱門依賴**透明（C）+ 教學 UI**，而非**隱藏（A）再揭露** |

#### （對立・獨立）Balatro

| 維度 | 內容 |
|------|------|
| **呈現** | **撲克牌型規則極簡且固定（A 級基底）**；成長來自 Joker／Planet 等**購買即全文可讀**的加成 |
| **玩家回饋（熱門相關）** | **正面**：10 分鐘內進入「數字爆炸」爽感、無劇情負擔、重開快。**負面**：後期數字與組合過多可能暈（玩家仍買單，因 Roguelike 隨機性） |
| **與 A/B/C 距離** | 啟示：**底層規則可先極簡**，但**新效果仍建議一次給清**；與「鎖條文」不同 |

## 3. 研究問題 1：本作企劃（熟練度 → A/B/C）是否有熱門先例？

### 3.1 結論（針對上述五款）

| 判斷 | 說明 |
|------|------|
| **同構先例：無** | 五款皆**未**採用「同一張怪物卡：先無技能 → 熟練度達標 → 一行技能 → 再達標 → 完整規則」 |
| **部分類似（需寫清差異）** | 見下表 |

| 類似點 | 作品 | 與本作差異 | 玩家回饋是否支撐「熱門」 |
|--------|------|------------|------------------------|
| 名稱有 Mastery | Marvel Snap | **外觀／表情**，非規則；進度綁 Variant | 熱門**不**靠 Mastery；Mastery **爭議** |
| 全域簡化 | Pokémon TCG Pocket | 設計師改卡＋改規則，**非**玩家個人進度 | **是**熱門主因之一 |
| 局內逐步複雜 | 月圆之夜、Slay the Spire | Run 內**取得新卡**，每卡仍**一次給齊**說明 | **是**；靠透明 + 組合深度 |
| 基底簡、元件全文 | Balatro | 基底像 A，但每個 Joker **購買即 C** | **是**；靠爽感與數學深度 |

### 3.2 文獻／桌遊向補充（不在五款內，論文可註腳）

| 作品 | 機制 | 為何仍非本作同構 |
|------|------|------------------|
| **Shards of Infinity**（桌遊） | Mastery 軌解鎖卡上能力 | 實體桌遊進度，非數位「熟練度條」 |
| **Grand Archive**（TCG 規則） | Mastery 替換式玩家能力 | 玩家級 Mastery，非單卡三階文案 |

**論文可寫**：本作提案屬 **「數位 TCG 簡化門檻」+「單卡規則分階揭露」** 的交叉區，熱門竞品多佔其中**一條軸**，少數兩條交集。

## 4. 研究問題 2：熱門作品如何呈現技能？玩家為何喜歡？

### 4.1 歸納四種「熱門公式」（由五款抽象）

| 公式 | 代表 | 玩家喜歡的原因（回饋指標） | 本作可借鏡 |
|------|------|---------------------------|------------|
| **F1 極短可讀** | Marvel Snap | 單局 <5 分鐘可決；**一眼懂** On Reveal | **B 階段**應真的只有一行，且關鍵字可點 |
| **F2 透明即完整** | Slay the Spire、月圆之夜 | 選牌時**不怕資訊詐欺**；失敗可歸因於構築 | 若長期目標是深度，**C 必須可查**（圖鑑／長按） |
| **F3 設計師幫你刪複雜度** | Pokémon TCG Pocket | 不用讀 9000+ 卡；**敢進門** | 本作 **A（無技能）** 與此同方向，但範圍更小（僅怪物） |
| **F4 簡基底 + 全文元件** | Balatro | 先會玩，再追組合；**爽感**驅動重開 | 對戰核心先 ATK/HP；**法術／場地**可當「元件」 |

### 4.2 玩家**不喜歡**什麼（與熱門度負相關的訊號）

| 訊號 | 出處 | 對本作 A/B/C 的警示 |
|------|------|---------------------|
| 「要刷 Variant 才有 Mastery」 | Marvel Snap 社群 | **熟練度若像課金牆**會被反感；應綁**對戰行為** |
| 「多貨幣、看不懂」 | Pokémon TCG Pocket 評論 | 分階揭露**不能**再疊太多系統皮 |
| 「卡敘述歧義」 | Slay the Spire 少數負評 | **B 一行**必須與 **C 全文** 語意一致，避免升階後「規則變了」 |

## 5. 本作 A／B／C 與五款的对照矩陣

|  | **A 無技能** | **B 一行** | **C 完整** | 五款多數落在 |
|--|-------------|-----------|-----------|--------------|
| **本作提案** | ✓ 預設 | ✓ 熟練度 1 | ✓ 熟練度 2 | — |
| Pocket | 部分（極簡局內規則） | ✓ 改寫能力 | 實體版才有完整 | B（全域） |
| 月圆之夜 | — | 少數關鍵字卡 | ✓ | C |
| Marvel Snap | — | ✓ 常態 | 詳情靠關鍵字 | B |
| Slay the Spire | — | — | ✓ | C |
| Balatro | ✓ 撲克基底 | — | ✓ 每張 Joker | A 基底 + C 元件 |

### 5.1 對本作企劃的建議（仍不實作，僅策略）

| 建議 | 理由 |
|------|------|
| **A 可保留為預設體驗** | 與 Pocket、Balatro 的「簡基底」同賽道驗證過市場 |
| **B 必須「玩了就生效」或明確標「僅預覽」** | Snap Mastery 失分在「進度與玩法無關」；B 若不解鎖異能會被罵 |
| **C 用「可查」而非「強迫對戰中閱讀」** | StS／月圆之夜成功靠**玩家主動選牌時閱讀** |
| **論文敘事** | 定位為 **「Pocket 式門檻 + StS 式透明（延後）」** 的實驗性混合，**非** Snap 式 Mastery |

## 6. 建議驗證：本作三階 vs 竞品印象（給組員試玩／問卷）

### 6.1 材料

同一張怪物卡準備三版卡面（僅改技能區）：

- **A**：僅 ATK／HP  
- **B**：+ 一行「戰技摘要」  
- **C**：+ 完整條文（或長按開 C）

### 6.2 問題（7 點量表 + 開放題）

1. 哪一版你最願意**第一場**就玩？（A/B/C）  
2. 哪一版你最願意**玩超過 10 場**後還用？  
3. 「鎖著不說清楚」會不會降低信任感？（呼應 Snap Mastery 反感）  
4. 「一行摘要」是否足夠做決策？（呼應 Snap／Pocket）  
5. 你是否願意為「解鎖完整說明」多打幾局？（驗證熟練度動機）  
6. 對照你玩過的一款：**月圆之夜／Slay the Spire／Snap／Pocket／Balatro** 最接近哪一版？  
7. 開放題：請用一句話描述你喜歡的那款「卡牌效果是怎麼給你的」。

### 6.3 結果怎麼寫進論文

| 若結果… | 論文結論方向 |
|---------|--------------|
| 多數喜歡 A→B，但要求 B 即生效 | 支持分階，反對「純鎖文案」 |
| 多數直接要 C | 熱門公式偏向 StS／月圆之夜；本作 A 只適合教學關 |
| 多數喜歡 A 且厭惡鎖 | 支持無技能，**反對**熟練度解鎖；改為 Pocket 式全域簡化 |
