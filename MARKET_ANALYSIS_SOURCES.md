## 1. 收錄標準（本目錄怎麼選）

### 1.1 必須符合

| 條件 | 說明 |
|------|------|
| **已商業化** | 曾或現於 Steam／手機商店／主機等**付費或 F2P 正式營運**（含 Early Access 轉正式） |
| **獨立性** | 以**小團隊／獨立發行商**為主；非暴雪、Riot、萬代等 3A 常規 TCG 管線 |
| **可查到一手來源** | 至少有 Steam 商店頁、官網、或發行新聞稿之一 |

### 1.2 刻意不收（但論文可當「產業背景」一句帶過）

| 類型 | 範例 | 原因 |
|------|------|------|
| 3A／大廠常規 TCG | 《爐石傳說》《Marvel Snap》《闇影詩章》手遊版 | 非獨立；預算與營運模式不可比 |
| 純實體 TCG | 萬智牌紙牌、寶可夢 PTCG | 非數位獨立遊戲賽道 |
| 僅 Demo／Jam | 多數 Itch 48h 原型 | 未商業化 |

### 1.3 類型三分法（本目錄欄位 `類型`）

| 代碼 | 名稱 | 定義 |
|------|------|------|
| **TCG** | 集換式／構築牌組對戰 | 明確「牌組構築 + 1v1（或多線）對戰」；可含 F2P 開包 |
| **旁支** | 卡牌旁支 | 牌組／卡牌驅動，但核心循環是 Roguelike、自走棋、背包整理、戰棋等 |
| **對立** | 類型對照 | **不以卡牌構築為核心**的商業化獨立作；用來對照「為何要做／不做 TCG」 |

### 1.4 地區二分（工作定義）

| 分類 | 定義（務實用，非政治學精確） |
|------|------------------------------|
| **亞洲** | 開發商／發行商總部或主體團隊常見於 **東亞、東南亞、南亞**（含中國大陸、香港、台灣、新加坡、日本、韓國等） |
| **非亞洲** | 歐洲、北美、大洋洲、拉美等 |

> 若工作室跨國（例：新加坡總部 + 中國團隊），表中會註明**「亞洲（新加坡）」**並在備註寫雙邊背景。

## 2. 建議使用的二級資料來源（怎麼搜、怎麼引用）

### 2.1 商店與數據

| 來源 | 網址 | 適合查什麼 |
|------|------|------------|
| **Steam 商店** | https://store.steampowered.com | 類型標籤、評價比例、語言、DLC、EA 歷程 |
| **SteamDB** | https://steamdb.info | 發售日、價格歷史、同時在線（若公開） |
| **Steam 標籤排行** | 商店 → 瀏覽 → 標籤 `Deckbuilding` / `Card Game` / `Trading Card Game` | 同賽道密度與玩家用語 |
| **Google Play / App Store** | 各商店 | 亞洲手機獨立卡牌的評分、下載量區間、課金描述 |
| **IsThereAnyDeal / MetaCritic** | 彙總評分 | 跨平台口碑摘要 |

### 2.2 產業與新聞（亞洲／全球）

| 來源 | 網址 | 適合查什麼 |
|------|------|------------|
| **QooApp News** | https://news.qoo-app.com | 亞洲手遊／獨立上 Steam 消息（中英） |
| **GamesPress** | https://www.gamespress.com | 發行通稿、上架日期 |
| **MobyGames** | https://www.mobygames.com | 開發商國別、作品年表 |
| **itch.io 發行商頁** | 例：https://erabitstudios.itch.io | 獨立發行商作品集 |

### 2.3 設計與趨勢（論文「文獻回顧」用）

| 來源類型 | 建議關鍵字（中英） |
|----------|-------------------|
| 學位／研討會 | `digital TCG` `deckbuilding roguelike` `complexity` `onboarding` |
| GDC／演講 | `Slay the Spire` design postmortem；`Marvel Snap` accessibility（若對比複雜度） |
| 玩家社群 | Steam 評論篩選「新手」「複雜」「Pay to win」；Reddit r/digitalcardgames、NGA／巴哈姆特（繁中） |

### 2.4 本專題分析時建議記錄欄位

對每一款竞品建議填：**平台、商業模式、單局長度、規則揭露方式、長期成長軸、與熟練度提案的距離（高／中／低）**。

## 6. 依「與本作距離」的速查矩陣（給熟練度提案用）

| 研究問題 | 建議優先讀 |
|----------|------------|
| 華人獨立、真的叫 TCG 的有哪些？ | **回纹集换卡牌**、**Duel Champions**（未來）、**Nova Island**（F2P） |
| 亞洲獨立如何把牌組變「好上手」？ | **Indies' Lies**、**Looper Tactics**、**MetaCard** |
| 獨立 TCG 為何難活？ | **Mythgard** 沿革、Steam 評論「圈小／匹配／課金」 |
| 不做 TCG 的牌類爆款？ | **Slay the Spire**、**Backpack Battles**、**Balatro** |
| 規則分階揭露有無先例？ | **Cardpocalypse**（進度改規則）、**Inscryption**（敘事藏規則）；**Marvel Snap Character Mastery**（精通多為**外觀**，非規則解鎖——若寫論文需註明非獨立） |
| 與本作「訓練廳、場地效果、無怪獸技能」最近？ | 少有完全同構；需以 **旁支** 為主、**TCG** 為輔 |
