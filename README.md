# 卡牌遊戲（Unity 專題）

本說明以**玩家實際操作時能體驗到的功能**為主；部分內容仍在擴充中，以遊戲內實際畫面為準。

## 執行環境

- **Unity Editor**：2023.1.22f1（建議使用相同或相容版本開啟專案）
- **平台**：於編輯器 Play 或建置後執行（依本機環境為準）

---

## 玩家能體驗的功能

### 登入與存檔槽

- 進入 **登入場景**後，可選擇**既有玩家槽位**或**建立新玩家**。
- 支援多個玩家槽位配置；可依介面流程管理個別紀錄（實際選項以場景按鈕為準）。

### 大廳（首頁）

- 進入 **大廳**後作為主要落點，可由此前往其他功能場景（商店、編組、對戰等，依場景配置為準）。

### 全域導航（畫面上的「≡」）

多數非登入場景可開啟**左上選單**，通常包含：

- **回首頁**：返回大廳。
- **玩家資訊**：查看／編輯與個人紀錄相關的摘要資訊。
- **背包**：進入與持有物相關的場景（Persistent）。
- **遊戲設定**：進入 **Settings** 場景，可調整**顯示比例（戰鬥場景套用預設）**、**畫面品質**，並查看參數對照（實際選項以選單為準）。
- **回到登入頁面**：返回登入流程。

### 玩家資訊

- 可檢視如 **識別資訊、角色稱呼、金幣、牌組摘要、英雄摘要、對戰統計（勝負與場次等）** 等（實際欄位以介面顯示為準）。
- 於 Play 模式下開啟玩家資訊時，專案會將部分資料寫入 `Assets/PlayerDataSnapshots/` 下的快照檔（供開發／除錯追溯）。

### 牌組編輯（Buildbeck）

- 在 **組牌／編輯牌組**場景中，可瀏覽卡牌並編輯牌組。
- **左側館藏格（Library Grid）**：列表上顯示的是**組牌用縮圖**（與專案內 `DeckThumb` 美術一致），與商店／詳情中較大的**本體立繪**（`CardArt`）可視為不同用途的圖檔；細節見 [GAMEPLAY_AND_RULES.md](./GAMEPLAY_AND_RULES.md)。
- **多個牌組槽位**：可切換不同槽位進行編輯。
- **儲存牌組**：將目前編輯結果寫入存檔。
- **解散／清空牌組**：依按鈕功能重置該槽牌組（請留意是否會清空該槽內容）。
- **編輯牌組名稱**：可為目前槽位設定顯示名稱；若場景有對應標籤，會顯示目前槽位的名稱。

### 遊戲設定（Settings）

- 經由全域選單進入 **Settings** 場景。
- **顯示比例**：可選擇戰鬥場景的卡牌顯示調校**預設**（例如「預設一／對照組」）；選擇後會套用並可檢視**參數對照**（手牌／場上大小與文字等摘要）。
- **畫面品質**：可切換 Unity **Quality** 等級（會一併記住選擇）。
- 實際套用與存檔鍵請見 [GAMEPLAY_AND_RULES.md](./GAMEPLAY_AND_RULES.md) 的 **第五節**（Settings）與 **第四節 4.5**（對戰顯示比例 JSON）。

### 卡片商店

- 進入 **商店場景**後可進行與取得卡片相關的操作（購買規則與庫存以遊戲內為準）。

### 對戰

- 進入 **對戰／模擬對戰**相關場景後，可進行回合制卡牌對戰。
- 對戰中包含 **手牌、出牌、場上單位、回合流程** 等；並有 **天氣／環境類效果** 影響局勢（名稱與規則以遊戲內說明為準）。
- 部分場景可能提供 **除錯／批次模擬**相關 UI（開發用），一般遊玩可依需求忽略。

### 主線／劇情

- 設有 **主線場景**入口（內容長度與章節依專題進度而定）。
