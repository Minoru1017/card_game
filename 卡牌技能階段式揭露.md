## 2. 三階揭露速查（與熟練度 GDD 對齊）


| 階段    | 卡面（對戰）                | 圖鑑                    | 效果是否結算（建議）           |
| ----- | --------------------- | --------------------- | -------------------- |
| **A** | 無技能正文；可顯示 🔒「熟練後解鎖戰技」 | 「未解放」+ **模糊句**（不寫數值）  | **否**（僅 ATK／HP）      |
| **B** | **一行戰技摘要**            | 摘要 +「詳情未解放」           | **是**（摘要即為可玩規則）      |
| **C** | 保留 B 一行；可開「？」詳情       | **完整條文**（時機／對象／名詞／疊加） | **是**（與 B 同一效果，補足誤讀） |


**撰寫原則**

1. **B 與 C 效果一致**，C 只補「何時、對誰、疊加順序」，避免升階後規則突變。
2. **B 一行建議 ≤ 28 全形字**（約兩行 14 字卡面）。
3. 新名詞（如「直擊」）在 B 可用，在 C **必須定義**。
4. 高稀有度／高面板卡，戰技宜**補定位**（保護、控場），避免再堆攻擊。

**可玩 v1 代表卡（國王／王后／民兵）**  
背包檢視、對戰手牌長按等處採**無標點、空格分節**之規則句（與國王語氣一致）；**以 `MonsterSkillRegistry` 內字串為準**。下列 §4～§6 之 **B 一行**與 **C 完整條文** 為圖鑑／熟練度正式稿；v1 畫面暫以程式內 **B 摘要**＋**完整內文** 顯示（`ShowSkillLineBInUi` 目前關閉，長按顯示完整內文）。

## 3. 新卡填表模板


| 欄位                       | 說明                               |
| ------------------------ | -------------------------------- |
| `id` / 卡名 / 稀有度 / ATK／HP | 來自 `CardList.csv`                |
| **A·圖鑑模糊句**              | 不劇透機制，可帶世界觀                      |
| **B·戰技名 + 一行**           | 玩家對戰可見、可結算                       |
| **C·完整條文**               | 圖鑑／長按；含■何時 ■效果 ■名詞 ■備註           |
| **A→B 熟練度**              | 見 `CARD_PROFICIENCY_GDD.md` §4.3 |
| **B→C 熟練度**              | 同上                               |
| **備選方案**                 | 進攻／防守互換等                         |

## 4. 已定案範例：國王

### 4.1 卡牌資料


| 項目       | 內容                                                 |
| -------- | -------------------------------------------------- |
| **id**   | 13                                                 |
| **卡名**   | 國王（King）                                           |
| **稀有度**  | UR                                                 |
| **面板**   | ATK **80**／HP **80**                               |
| **設計取向** | 高面板**訓練廳指揮官**；戰技保護英雄、處理**直擊**，不再加攻擊避免 UR 過強        |
| **關聯卡**  | 王后（id 12）、王國騎兵（id 6）、城堡（id 7）——**王室連攜**留待日後，本範例不啟用 |


> **數值為企劃草案**；實作前須對照 `BattleSimulationManager` 結算與 UR 曲線調整。

### 4.2 階段文案

#### A 階段（未解放）


| 位置      | 文案                    |
| ------- | --------------------- |
| 卡面技能區   | 🔒 **熟練後解鎖戰技**        |
| 圖鑑·模糊提示 | 據說能在訓練廳裡號令隊形保護己方導師 |


**結算**：無戰技；僅 80／80。

---

#### B 階段（基礎解放）

**戰技名**：庭訓號令  

**卡面一行**：

> 英雄受到的**直擊**傷害 −5（每場對戰至多 3 次）。

**結算**：**生效**（玩家應能實際觸發減傷）。

---

#### C 階段（完整解放）

**【戰技】庭訓號令（被動）**

**■ 何時檢查**  
我方英雄即將受到**直擊**傷害時——例如對方場上無怪獸、對方怪獸攻擊目標為我方英雄，或指定英雄的直擊類法術。

**■ 效果**  
該次直擊傷害 **−5**（減後至少仍造成 **1** 點傷害）。  
本局對戰內，此減傷最多生效 **3** 次；第 4 次起不再觸發。

**■ 名詞**  
**直擊**：不經雙方場上怪獸攻防互換、直接扣減英雄 HP 的傷害。

**■ 與場地效果**  
可與「**訓練薄霧**」（直擊英雄傷害 −50%）等場地效果疊加。  
結算順序：**先場地效果，再戰技減傷**（若實作順序不同，以程式與本文件同步修訂為準）。

**■ 圖鑑備註（設計用）**  

- 國王 UR 面板已偏高，戰技不提供額外攻擊。  
- 與**王后**同場時，本版**無**額外連攜；未來可於熟練度或 DLC 另訂「王室連攜」。

**結算**：與 B 相同效果；C 供查規則與爭議裁定。

---

### 4.3 熟練度門檻（與全卡共通）

| 階段 | 條件 |
|------|------|
| B | 牌組含本卡且累計 **5 勝**（不限難度） |
| C | 牌組含本卡且累計 **10 勝**（**普通以上**難度） |
| 御三家 | 國王／王后／民兵預設 **C**（見 GDD §4.1） |

---

### 4.4 備選方案：進攻型國王

若測試或問卷顯示玩家期待 UR「能壓制對面英雄」，可改採下表（**與 §4.2 防守型二擇一**，勿同時實装）。


| 階段    | 備選文案                                                     |
| ----- | -------------------------------------------------------- |
| **B** | **王者駕臨**：你對敵方英雄造成的**直擊**傷害 +5（每場對戰至多 3 次）。               |
| **C** | 於 B 基礎上補充：直擊含「對方場上無怪獸時」的怪獸攻擊與直擊類法術；與對方「訓練薄霧」可同時影響雙方直擊結算。 |

## 5. 已定案範例：王后

### 5.1 卡牌資料


| 項目       | 內容                                      |
| -------- | --------------------------------------- |
| **id**   | 12                                      |
| **卡名**   | 王后（Queen）                               |
| **稀有度**  | SR                                      |
| **面板**   | ATK **20**／HP **40**                    |
| **設計取向** | 中面板**生存型**；戰技吸收首次爆擊，適合訓練廳試誤       |
| **關聯卡**  | 國王（id 13）——本版**無**王室連攜；連攜留待日後 |


### 5.2 階段文案

#### A 階段（未解放）


| 位置      | 文案                      |
| ------- | ----------------------- |
| 卡面技能區   | 🔒 **熟練後解鎖戰技**          |
| 圖鑑·模糊提示 | 據說能在危急時為王室撐起第一道護盾 |


**結算**：無戰技；僅 20／40。

---

#### B 階段（基礎解放）

**戰技名**：王室庇護  

**卡面一行**：

> 王室庇護 首次減3 最少1 全場1次 之後不再觸發

**結算**：**生效**（本局首次受傷 −3，減後至少 1）。

---

#### C 階段（完整解放）

**【戰技】王室庇護（被動）**

**■ 何時檢查**  
此**王后**即將受到傷害時（含怪獸戰鬥、法術等；**不含**無傷害的判定）。

**■ 效果**  
本局對戰內**第一次**觸發時，該次傷害 **−3**（減後至少仍造成 **1** 點）。  
**第二次起**本戰技不再觸發。

**■ 與國王**  
與國王同場時，本版**無**額外連攜。

**結算**：與 B 相同效果；C 供查規則與爭議裁定。

---

### 5.3 可玩 v1 定案字串（程式對照）


| 用途        | 定案文案 |
| --------- | ---- |
| **B 摘要**  | `王室庇護 首次減3 最少1 全場1次 之後不再觸發` |
| **完整內文**  | `首次受到傷害時 王后減傷3點 最少1點 對戰內僅1次 其後不再觸發` |
| **戰報**    | `王室庇護：這次傷害少 3 點 本局不再觸發` |


**程式**：`MonsterSkillIds.Queen` · `MonsterSkillRegistry.ApplyQueenShelter`

---

### 5.4 熟練度門檻

同 §4.3（王后非御三家，需累計勝場）。


| 轉換        | 條件（草案）                                    |
| --------- | ----------------------------------------- |
| **A → B** | 王后曾納入牌組，並完成 **3** 場對戰；其中至少 **1** 場曾將王后置於場上 |
| **B → C** | 使用王后的對戰累計 **2** 勝；**或** 本戰技已成功觸發 **1** 次     |

## 6. 已定案範例：民兵

### 6.1 卡牌資料


| 項目       | 內容                            |
| -------- | ----------------------------- |
| **id**   | 4                             |
| **卡名**   | 民兵（Militia）                   |
| **稀有度**  | N                             |
| **面板**   | ATK **20**／HP **20**          |
| **設計取向** | 低費試玩卡；戰技獎勵**首次上場**，教導「置場時機」   |
| **關聯卡**  | 無                             |


### 6.2 階段文案

#### A 階段（未解放）


| 位置      | 文案                    |
| ------- | --------------------- |
| 卡面技能區   | 🔒 **熟練後解鎖戰技**        |
| 圖鑑·模糊提示 | 據說能在列隊上場時短暫提振陣腳 |


**結算**：無戰技；僅 20／20。

---

#### B 階段（基礎解放）

**戰技名**：列陣  

**卡面一行**：

> 列陣 首次置場 攻擊+5 全場1次 留至離場

**結算**：**生效**（本局首次將此民兵置於場上時 ATK +5）。

---

#### C 階段（完整解放）

**【戰技】列陣（觸發）**

**■ 何時**  
本局對戰內，**第一次**將**此民兵**由手牌置入我方場上時。

**■ 效果**  
該民兵 **攻擊力 +5**。  
加成持續至該民兵**離場**或**對局結束**。

**■ 次數**  
每局對戰**僅觸發 1 次**（同一張民兵實例；若離場後再次上場，本版**不**再次 +5）。

**結算**：與 B 相同效果；C 供查規則與爭議裁定。

---

### 6.3 可玩 v1 定案字串（程式對照）


| 用途        | 定案文案 |
| --------- | ---- |
| **B 摘要**  | `列陣 首次置場 攻擊+5 全場1次 留至離場` |
| **完整內文**  | `首次置於場上時 民兵攻擊力+5點 對戰內僅1次 加成留至該民兵離場或對局結束` |
| **戰報**    | `列陣：這次攻擊力多 5 點 本局僅1次` |
| **Toast** | `民兵·列陣：攻擊力多 5 點` |


**程式**：`MonsterSkillIds.Militia` · `BattleSimulationManager.ApplySummonMonsterSkills`

---

### 6.4 熟練度門檻

民兵為**御三家**，預設 **C**；其餘同 §4.3。


| 轉換        | 條件（草案）                                      |

| **A → B** | 民兵曾納入牌組，並完成 **2** 場對戰；其中至少 **1** 場曾將民兵置於場上 |
| **B → C** | 使用民兵的對戰累計 **1** 勝；**或** 本戰技已成功觸發 **1** 次       |
